## Learned User Preferences

- Keep guidance practical and concise for non-technical recipients, especially installation or release instructions.
- When reviewing MedHub automation changes, stress-test against the live DOM and call out real replay gaps before summaries.

## Learned Workspace Facts

- `auto-mate` is a Chrome extension that records MedHub procedure form steps and replays them from spreadsheet rows using the user's logged-in browser session.
- GitHub releases attach `auto-mate-extension.zip`; users install it by unzipping, keeping the `extension` folder, loading it unpacked in Chrome, then reloading the extension after updates.
- MedHub procedure logging support centers on live fields such as `procedure_date`, `location_other`, supervisor `searchterms`, `patientID_other`, `procedures_searchterms`, optional demographics/notes fields, `procedure_submit`, and `log_another`.
